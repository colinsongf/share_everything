## 使用说明

### 物出

- 请输入'wuchu+物品ID+验证码'确认，验证码请向新物主查询。记得要输入+这个符号哦。

### 物入

- 如果您是物品所有权人，第一次录入物品，请输入'wuru+物品名称'；如果您是借用人，请输入'wuruid+物品ID'，物品ID请向原物主查询。

### 随便看看

- 请输入'sbkk'，会显示您的所有权和使用权信息。如果您没有任何物品信息，将会随机显示5条系统中的物品信息供您参考。
